# Contributing to sbom-vm

Thanks for your interest! To contribute:

- Read the README and AGENTS.md (if present).
- Open an issue to discuss larger changes before implementing.

Local development
- The project is a small Python script. Ensure Python 3.8+ is installed.
- Edits should be small and focused. Run `python3 -m unittest discover` if tests exist.

Commits and pull requests
- Create a descriptive branch: `repo-assist/feature-...` or `repo-assist/fix-...`.
- Commit with clear messages and include a Co-authored-by trailer for automated commits.
- Push and open a draft PR; include tests when adding functionality.

Code of conduct
- Be respectful and follow the project's norms.

Maintainers: please review small contributions promptly to keep momentum.
